[spring] SpEL

发表于 2019-10-06 更新于 2019-12-18 分类于 java ， spring

用途

SpEL的特性：

  • 使用bean的ID来引用bean。
  • 调用方法和访问对象的属性。
  • 进行运算。
  • 正则表达式匹配。
  • 集合操作。

可用来在依赖注入以外的地方。例如：Spring Security支持使用SpEL表达式定义安全限制规则。
