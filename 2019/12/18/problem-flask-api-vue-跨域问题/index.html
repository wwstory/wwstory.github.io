[problem] flask api + vue 跨域问题

发表于 2019-12-18 分类于 problem

问题

使用flask提供json数据，用vue在前端获取json数据展示。

发现无法获取到该json数据，但其它网站的api的json数据是可以拿到的。

解决

后来，查到是由于遇到跨域请求的问题，通过设置flask允许其它域访问即可。

域 = 协议名 + 主机名 + 端口号

禁止不同域的访问是为了防止CSRF(跨站请求伪造)攻击。

通过CORS(跨源资源共享)允许的请求。
